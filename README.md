# csphackathon
CSP Hackathon server templates.

## Installation
Requirements: [bun](https://bun.sh).
```sh
# Install all dependencies
bun i

# Start the server
bun start
```

## Project structure
```
csphackathon/
└── src/
    ├── config/       # Contest configurations
    │   └── problems/ # Problems and checkers
    ├── views/        # HTML utilities
    └── routes/       # Pages and API routes
```
